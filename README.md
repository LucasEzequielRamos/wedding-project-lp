# Wedding Landing Page — Lu & Pri
A modern and interactive wedding landing page designed to provide guests with all the essential information about thLive Demo: https://ourweddinglupri.vercel.app/
---
IMPORTANT:
This project is designed for mobile view only.
It is optimized for smartphones and may not display correctly on desktop screens.
---
##Features
- Interactive Wedding Page — Centralized information for guests
- Event Details — Location, schedule, and important data
- Visual Experience — Images and sections designed to tell a story
- Mobile-First Design — Fully optimized for mobile devices
- Fast Performance — Lightweight and optimized frontend
---
##Tech Stack
Framework: Next.js
Language: TypeScript
Styling: Tailwind CSS
Deploy: Vercel
---
##Architecture Highlights
- Mobile-First Approach — Designed primarily for smartphone interaction
- Component-Based UI — Reusable components for scalability
- Clean Layout Structure — Focused on readability and user flow
- Optimized Performance — Minimal assets and efficient rendering
---
##Getting Started
##Prerequisites:
- Node.js 18+
##Installation:
git clone https://github.com/LucasEzequielRamos/wedding-project-lp.git
cd wedding-project-lp
npm install
Run the App:
npm run dev
